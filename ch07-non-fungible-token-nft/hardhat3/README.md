
## Project Overview – ERC721 NFT (Spacebear)


This project demonstrates deploying and testing an **ERC721 NFT** contract called **Spacebear** using **Hardhat 3**, TypeScript, and Viem. Includes local tests and deployment to the **Sepolia testnet**.


What’s implemented (Hardhat)

ERC721 NFT using OpenZeppelin contracts

Owner-restricted minting (Ownable)

Token URI storage per NFT

Local unit tests with Hardhat + Viem

Deployment script for testnet

Deployment and source verification on Sepolia


### Deployed Contract

- Network: Sepolia Testnet
- Contract Name: Spacebear
