
# Smart Contract Wallet

My Solidity practice for implementing a Smart Contract Wallet.



### Requirements implemented

- Single wallet owner
- Can receive ETH in all cases
- Owner can send ETH to any address (EOA or contract)
- Allowance system for limited spending by other addresses
- Social recovery: owner can be changed by 3 out of 5 guardians



### Files / Iterations

- SmartContractWallet.sol: My implementation
- SmartContractWallet2.sol: Code presented in the course.
